# 🛒 쇼핑 리스트 앱 (Shopping List App)

브라우저에서 바로 실행되는 간단한 쇼핑 리스트 웹 앱입니다. 별도의 빌드나 서버 없이 `index.html` 하나로 동작합니다.

## 기능

- 항목 추가 / 삭제
- 체크박스로 구매 완료 표시 (취소선 처리)
- 완료된 항목 한 번에 지우기
- 전체 개수와 남은 개수 표시
- `localStorage`에 자동 저장 — 새로고침해도 목록 유지

## 실행 방법

`index.html` 파일을 브라우저에서 열면 됩니다.

## 기술 스택

- HTML / CSS / Vanilla JavaScript (프레임워크 없음)
